# Modular Arithmetic

● Modulus M에 도달하면 다시 0 으로 돌아가는 정수들을 가지고 하는 연산<br>
● 모듈러 연산에서 모든 정수는 M으로 나눈 나머지의 값으로 표현된다.<br>

### 만약 시계의 시간이라고 했을때

```

~~25시~~ --> 1시 <br>
~~42시~~ --> 18시

```

## 구성 절차 (알고리즘 문제 해결 曰)

1. 일종의 합동식하고도 연관이 있는데
   a,b를 M으로 나눈 나머지가 각각 a',b'이라고 한다면, a= XM+a',b=YM+b' a+b = (X+Y)M +(a'+b') M으로 나눈 나머지는 (a'+b') mod M 의 꼴이 된다.<br>
